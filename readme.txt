プログラム作品（プラグイン/アプリ/ツール）を登録・公開。GitHub連携、CSV入出力、REST API、自動更新、ショートコード、色設定、コピー用ボタン対応。

== Features ==
* Custom Post Type: icpw_prog
* Meta fields with REST write support
* REST API: /wp-json/icpw/v1/works (GET/POST/PUT/DELETE)
* CSV Import/Export: Tools -> Program Works CSV
* GitHub helper (latest tag/license/default branch)
* GitHub Releases-based auto update
* Shortcode: [icpw_works] (一覧カード + GitHub情報 + コピー用ボタン)
* Settings: Settings -> Program Works 設定 (色/表示切替)
